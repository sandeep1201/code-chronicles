# Your Post Title

Start writing your blog post content here...

Use this template as a starting point for new draft posts.

## Frontmatter Fields

### Required Fields

- **title**: The title of your blog post
- **slug**: URL-friendly identifier (lowercase, hyphenated version of title)
- **excerpt**: Brief description (150-160 characters for SEO)
- **scheduledPublishAt**: Date when post should be published (YYYY-MM-DD)
- **tags**: Array of relevant tags
- **author**: Author name
- **draft**: Must be `true` for drafts

### Optional Fields

- **publishedAt**: Will be automatically set to scheduledPublishAt if not provided
- **updatedAt**: Last update date (optional)
- **featured**: Whether to feature this post (default: false)

## Notes

- Posts are published weekly (one per week maximum)
- Scheduled date should be at least one week apart from other scheduled posts
- The system will automatically move the post from `drafts/` to the main `blog/` folder when published
- After publishing, `draft` will be set to `false` and `scheduledPublishAt` will be removed
