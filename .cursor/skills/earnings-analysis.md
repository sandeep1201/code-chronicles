# Earnings Analysis

## Description

Analyzes upcoming earnings reports using historical patterns, guidance trends,
and sector performance to estimate post-earnings moves.

## Prompt

Analyze upcoming earnings for [COMPANY_NAME].

Use:

- Historical earnings reactions (last 4-8 quarters)
- Guidance trends and management commentary
- Sector performance and peer comparisons
- Current sentiment (analyst ratings, options flow, price action)
- Pre-earnings positioning

Estimate:

- Most likely post-earnings move direction and magnitude
- Probability-weighted scenarios (bull case, base case, bear case)
- Key levels that could trigger larger moves
- Options strategies if applicable
- Risk factors specific to this earnings report

## Usage

Replace [COMPANY_NAME] with the company name or ticker. Use this skill before
earnings announcements to prepare for potential volatility and trading
opportunities.
