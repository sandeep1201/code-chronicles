# Staff Prep Daily Schedule (60 days)

Each day maps to Day N (1–60). Week = ceil(Day/5), DayInWeek = ((Day-1) % 5) + 1.

---

## Week 1: Assessment & Baseline

| Day | Focus | Tasks |
|-----|-------|-------|
| 1 | Self-assessment | List experience: years by language (Java, JS/TS), cloud, Kafka, React. Write honest self-assessment doc. |
| 2 | Coding baseline | 2–3 LeetCode Medium: arrays, strings, hashmaps (e.g. Two Sum, Valid Anagram, Group Anagrams). Note time and correctness. |
| 3 | System design baseline | 1 system design: "Design a URL shortener" (30–45 min). Note structure and depth. |
| 4 | STAR stories | Draft 5 STAR stories: leadership, conflict, failure, technical decision, impact. 1–2 pages each. |
| 5 | Java review | Review Java basics if rusty: collections, concurrency, JVM. List gaps. |

---

## Week 2: Arrays, Strings, Hashmaps, Linked Lists, Stacks

| Day | Focus | Tasks |
|-----|-------|-------|
| 6 | Arrays: two pointers | 2 problems (e.g. Two Sum II, Container With Most Water, 3Sum). NeetCode Arrays. |
| 7 | Arrays: sliding window | 2 problems (e.g. Best Time to Buy/Sell, Longest Substring Without Repeating). |
| 8 | Strings | 2 problems: parsing, anagrams, palindromes (e.g. Valid Palindrome, Valid Anagram). |
| 9 | Hashmaps | 2 problems: frequency, two-sum variants (e.g. Two Sum, Group Anagrams, Top K Frequent). |
| 10 | Linked lists | 2 problems: reversal, cycle detection, merge (e.g. Reverse Linked List, Linked List Cycle, Merge Two Sorted Lists). |

---

## Week 3: Stacks, Queues, Trees

| Day | Focus | Tasks |
|-----|-------|-------|
| 11 | Stacks & queues | 2 problems (e.g. Valid Parentheses, Implement Queue using Stacks). |
| 12 | Binary trees: traversal | 2 problems: BFS, DFS, max depth (e.g. Max Depth of Binary Tree, Level Order Traversal). |
| 13 | BST | 2 problems: search, validate, LCA (e.g. Validate BST, Lowest Common Ancestor of BST). |
| 14 | Graphs | 2 problems: BFS, DFS, shortest path (e.g. Number of Islands, Clone Graph). |
| 15 | Recursion & backtracking | 2 problems (e.g. Subsets, Combination Sum). |
| 16 | Review | 2 mixed problems from Week 2–3. Consolidate patterns. |
