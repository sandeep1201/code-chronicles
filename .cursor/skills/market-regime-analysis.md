# Market Regime Analysis

## Description

Determines the current market regime (trending, ranging, risk-on, or risk-off)
and provides guidance on how to adapt trading strategies accordingly.

## Prompt

Determine the current market regime: trending (bullish/bearish), ranging,
risk-on, or risk-off.

Include:

- Evidence supporting the regime classification
- How trading strategies should adapt in this environment
- Common mistakes traders usually make in this regime
- Key indicators to watch for regime changes

## Usage

Use this skill to understand the broader market environment before making
trading decisions. This should be one of the first analyses you run each day.
