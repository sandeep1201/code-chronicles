# Individual Stock Analysis

## Description

Provides comprehensive analysis of a specific stock by combining technical
structure, fundamentals, and recent news.

## Prompt

Analyze [STOCK_NAME/TICKER] by combining:

- Technical structure (chart patterns, support/resistance, trend analysis)
- Fundamentals (earnings growth, valuation metrics, financial health)
- Recent news and catalysts (earnings, guidance, sector developments)

Provide:

- Clear bullish or bearish bias with confidence level (high/medium/low)
- Key price levels to watch (entry, stop loss, targets)
- Time frame analysis (short, medium, long-term outlook)
- What type of trader this setup best suits (day trader, swing trader, position
  trader)

## Usage

Replace [STOCK_NAME/TICKER] with the actual stock symbol or company name. Use
this skill to deep dive into specific stocks identified through opportunity
identification or from your watchlist.
